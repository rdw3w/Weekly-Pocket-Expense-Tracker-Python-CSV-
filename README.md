💰 Weekly Pocket Expense Tracker (Python + CSV + Streamlit)

This project is a simple yet powerful Pocket Expense Tracker that helps you manage your weekly expenses with ease.

Add income & expenses using Python

Store data securely in CSV files

Visualize your spending with a Streamlit Dashboard

Beginner-friendly project to learn File Handling, CSV Module, and Basic Logic Building in Python

✨ Features

✅ User Authentication – Sign Up & Login system (stored in user.csv)
✅ Dashboard Interface – Built with Streamlit for a clean UI
✅ Expense Management – Add, view, and track daily/weekly expenses (stored in expenses.csv)
✅ Data Visualization – Charts & graphs to analyze where your money goes
✅ Beginner-Friendly – Great mini-project to understand Python + Streamlit + CSV handling

🛠️ Tech Stack

Python 3 🐍

Streamlit 🎨 (for UI and localhost dashboard)

CSV Files 📂 (for storing user data & expenses)

Basic CSS ✨ (for simple UI styling)

📂 Project Structure
📦 Weekly-Pocket-Expense-Tracker
 ┣ 📜 app.py                # Main Streamlit application
 ┣ 📜 user.csv              # Stores user signup/login data
 ┣ 📜 expenses.csv          # Stores expenses data
 ┣ 📜 requirements.txt     # Required libraries
 ┗ 📜 README.md             # Project documentation

▶️ How to Run

Clone this repository

git clone https://github.com/rdw3w/Weekly-Pocket-Expense-Tracker-Python-CSV-/tree/main


Navigate to the project folder

cd weekly-pocket-expense-tracker


Install dependencies

pip install -r requirements.txt


Run the Streamlit app

streamlit run app.py
